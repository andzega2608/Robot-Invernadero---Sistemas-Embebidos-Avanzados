# Robot Invernadero - Sistemas Embebidos Avanzados
El Robot Invernadero Inteligente es un sistema embebido avanzado diseñado para automatizar el proceso de monitoreo y cuidado de plantas dentro de un invernadero.
Su función principal es detectar las condiciones de las plantas (como humedad, temperatura) mediante sensores y rociar plaguicida automáticamente cuando sea necesario por medio de la lectura de un código QR.

La comunicación y el control remoto del sistema se realizan mediante el protocolo MQTT, lo que permite supervisar el estado del robot y enviar comandos desde un panel en Node-RED.
